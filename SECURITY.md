# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in TraceVault, please report it through GitHub's private vulnerability reporting feature.

**Do not open a public issue for security vulnerabilities.**

### How to Report

1. Go to the [Security tab](https://github.com/softwaremill/tracevault/security) of this repository
2. Click **"Report a vulnerability"**
3. Fill in the details of the vulnerability

We will acknowledge receipt within 3 business days and aim to provide an initial assessment within 10 business days.

## Supported Versions

Security updates are provided for the latest release only.
