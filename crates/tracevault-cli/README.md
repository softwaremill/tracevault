# tracevault

CLI tool for [TraceVault](https://github.com/softwaremill/tracevault) — AI code tracing and attribution.

## Install

```sh
cargo install tracevault-cli
```

## Usage

```sh
tracevault init        # Initialize in a repo
tracevault status      # Show tracing status
tracevault check       # Run attribution checks
tracevault push        # Push traces to server
```
